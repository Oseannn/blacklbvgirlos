# RetailOS

Application de gestion de point de vente (POS) moderne.

## Installation
Aucune installation nécessaire. Ouvrez simplement `index.html` dans votre navigateur.

## Identifiants de Connexion

- **Rôle**: Vendeuse

## Fonctionnalités
- **Admin Dashboard**: Vue d'ensemble, gestion des utilisateurs, rapports (mock), graphiques.
- **Vendeuse Dashboard**: Point de vente (POS) avec panier, calcul de taxes, gestion des clients.
- **Persistance**: Les données sont sauvegardées dans le `localStorage` du navigateur.

## Technologies
- HTML5 / CSS3 (Tailwind CDN)
- Vanilla JavaScript
- Material Symbols & Inter Font
